# Transcriber Pipecat

A real-time audio transcription project using Pipecat framework.

## Overview

This project provides real-time audio transcription capabilities using the Pipecat framework, which is designed for building voice and multimodal conversational AI applications.

## Features

- Real-time audio transcription
- Integration with Pipecat framework
- Support for various audio input sources
- Easy-to-use API

## Prerequisites

- Python 3.8+
- pip or poetry for package management

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```python
from transcriber_pipecat import TranscriberPipecat

# Initialize the transcriber
transcriber = TranscriberPipecat()

# Start transcription
transcriber.start()
```

## Configuration

Configure the transcriber by setting environment variables or passing configuration parameters.
